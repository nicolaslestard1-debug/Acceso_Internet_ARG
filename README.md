# Acceso_Internet_ARG
Mi primer código en Python! 

🌐 Análisis de la Brecha Digital y Conectividad

Este módulo se enfoca en la captura y tratamiento de datos provenientes de la Encuesta a Proveedores del Servicio de Acceso a Internet (INDEC). El objetivo es cuantificar las disparidades tecnológicas territoriales que condicionaron la escolarización virtual.

Estrategia Computacional: Implementación en Python utilizando Pandas para la limpieza y un diccionario de mapeo dinámico (mapeo_regiones_internet) para transformar datos provinciales en agregados regionales.


Visualización Escalamiento: Uso de bucles para la generación dinámica de gráficos de barras interanuales, aplicando escalas fijas en el eje Y para permitir una comparación directa y sin errores de interpretación.


Hallazgo Clave: Mientras que el Gran Buenos Aires alcanzó picos de casi 100 accesos por cada 100 hogares en 2020, regiones como el Noroeste (NOA) y el Noreste (NEA) apenas superaron los 40 accesos, evidenciando una brecha digital persistente.


💡 Reflexión

Como se detalla en el informe, el acceso a internet funcionó como una "desigualdad dinámica". La domiciliación escolar durante el confinamiento implicó que la escuela dejara de ser un espacio capaz de suspender el orden desigual , ya que la educación quedó supeditada a la infraestructura tecnológica preexistente en cada hogar
